Base64 Encoder / Decoder

Encode text to Base64 or decode Base64 back to readable text. Used everywhere from email attachments to API authentication and data URIs.

ⓘ Base64 is encoding, not encryption — anyone can decode it instantly. Never use it to "hide" passwords or sensitive data; use it only for safely transmitting binary data as text.
Encode (Text → Base64)
Decode (Base64 → Text)
Where Base64 shows up in real life: embedding images directly in HTML/CSS (data URIs), encoding email attachments (MIME), Basic Auth headers in HTTP requests, JWT tokens (the header and payload are Base64-encoded), and storing binary data in JSON or XML.
